    Hi There!

    So you want to roll your own feedback tab and you want to do it NOW! I've got just the thing. The full length documentation on this is pending internal review but the code is available today.

    What does this do?

    Right now this is a simple tab (like the feedback tab) that sits at the corner/side of your site and says "support". Click on it and poof! a page appears with some Lorem Ipsum (you obviously want to replace this with your own text. You can cancel or create a ticket. If you create a ticket it takes in the basic info and submits the new ticket to your Zendesk.
